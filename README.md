**Program for AAA protocol**
=====================

**Build project:**
-----------------------------------
```
$ ./build.sh
```

**Run project:**
-----------------------------------
```
$ ./run.sh [options]
```

**Test project:**
-----------------------------------
```
$ ./tests.sh
```
