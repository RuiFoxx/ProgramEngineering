**Program for AAA protocol**
=====================

**Build project:**
-----------------------------------

From Windows:
`> .\build.bat`

From Linux/OS X:
`$ ./build.sh` (will be added later)

**Run project:**
-----------------------------------

From Windows:
`> .\run.bat [options]`

From Linux/OS X:
`$ ./run.sh [options]` (will be added later)

**Test project:**
-----------------------------------

From Windows:
`> .\tests.bat` (will be added later)

From Linux/OS X:
`$ ./tests.sh`
